6. Analytics Workflow
Dataset Profile → Data Cleaning Checks → KPI Computation → Statistical Analysis → Correlation → Anomaly Detection → Structured Insight Objects → Persist Analytics Context
7. Insight Object Schema
Each insight should include:• Insight ID• Category (Trend, Anomaly, KPI, Correlation)• Evidence (metrics used)• Severity• Confidence• Human-readable summary• References to source columns
8. AntiGravity Master Prompt
Generate a production-grade Analytics Engine in FastAPI. Implement modular EDA, KPI computation, descriptive statistics, correlation analysis, anomaly detection and structured insight generation using Pandas/Polars. Expose REST APIs, persist analytical artifacts in PostgreSQL, write clean service-oriented code with unit tests, logging, type hints and documentation. Do not integrate an LLM; produce deterministic analytics_context JSON for later AI consumption.
