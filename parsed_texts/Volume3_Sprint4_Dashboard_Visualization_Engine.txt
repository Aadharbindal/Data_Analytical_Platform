7. UX Principles
Desktop-first enterprise design.
Maximum three-click navigation.
Progressive disclosure for complex analytics.
Consistent spacing, typography and colors.
Loading skeletons for async operations.
Accessible keyboard navigation.
8. AntiGravity Master Prompt
Generate a production-grade analytics dashboard using Next.js, Tailwind CSS, shadcn/ui and Plotly. Build reusable KPI cards, interactive charts, global filters, drill-down pages, responsive layouts, dataset explorer, insight panels, React Query integration, Zustand state management, loading skeletons, error boundaries and enterprise design patterns. Consume the analytics_context APIs without implementing AI chat.
9. Test Checklist
Dashboard renders analytics correctly.
KPI values match backend.
Charts update after filtering.
Responsive layout verified.
Accessibility audit completed.
Performance acceptable for large datasets.
