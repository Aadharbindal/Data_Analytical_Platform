7. Prompt Engineering Rules
Never expose raw datasets unless explicitly required.
Always ground outputs in analytics_context.
Separate facts, interpretations and recommendations.
Return structured JSON before natural-language formatting.
Include confidence and supporting evidence.
8. AntiGravity Master Prompt
Generate a production-ready AI service in FastAPI. Implement Context Builder, Prompt Builder, LLM Gateway, Response Validator, Recommendation Engine and Confidence Scoring. Use analytics_context as the only reasoning input, support multiple LLM providers through an abstraction layer, version prompts, validate responses against computed metrics, expose REST APIs, persist AI artifacts, implement retry logic, structured logging, comprehensive tests and clean architecture. Prevent hallucinations by grounding every insight in deterministic analytics.
