8. AntiGravity Master Prompt
Generate a production-ready monorepo for an AI Business Analyst Platform using Next.js, FastAPI, PostgreSQL, Redis and Docker Compose. Create modular folder structures, JWT authentication, SQLAlchemy models, Alembic migrations, workspace management, health endpoints, centralized configuration, structured logging, error handling, environment variable management and clean architecture principles. Do not implement analytics yet. Produce maintainable, enterprise-grade code with documentation and tests.
9. Acceptance Criteria
Application starts with one command.
Frontend connects to backend.
User registration and login work.
Database migrations succeed.
Health endpoint returns success.
Workspace creation is functional.
