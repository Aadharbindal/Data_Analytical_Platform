7. Report Design Principles
Lead with executive summary.
Include KPIs before detailed charts.
Reference evidence for every recommendation.
Support organization branding.
Preserve report versions for traceability.
Provide export in PDF and PPTX formats.
8. AntiGravity Master Prompt
Generate a production-ready reporting module in FastAPI. Build report generation, PDF and PowerPoint export, reusable templates, scheduled reporting, secure sharing, versioning, audit logging, report history, REST APIs and automated tests. Use analytics_context and validated AI insights as inputs. Design the system for enterprise branding, extensibility and high-quality executive outputs.
