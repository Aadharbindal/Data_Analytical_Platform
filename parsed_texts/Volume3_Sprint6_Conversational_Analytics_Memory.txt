7. Memory Rules
Remember only workspace-relevant context.
Retrieve analytical artifacts before calling the LLM.
Expire inactive sessions after configurable timeout.
Store conversation metadata separately from prompts.
Never fabricate historical answers.
8. AntiGravity Master Prompt
Generate a production-grade conversational analytics module using FastAPI. Build workspace-aware chat, conversation memory, retrieval over analytics_context, streaming responses, conversation persistence, suggested follow-up questions, evidence grounding, REST APIs, retry handling, structured logging and comprehensive tests. Ensure every response references deterministic analytical artifacts and supports future multi-agent orchestration.
